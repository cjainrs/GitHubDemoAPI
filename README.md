# GitHubDemoAPI

ASP.NET Core 8 Web API project created for learning Git and GitHub.

## Features

- ASP.NET Core 8 Web API
- Swagger Integration
- Git Branching
- Pull Requests
- Merge Conflict Resolution

## Technologies Used

- .NET 8
- ASP.NET Core Web API
- Swagger/OpenAPI
- Git
- GitHub

## Getting Started

### Clone Repository

```bash
git clone https://github.com/cjainrs/GitHubDemoAPI.git
```

### Navigate to Project

```bash
cd GitHubDemoAPI
```

### Run Project

```bash
dotnet run
```

### Open Swagger

```
https://localhost:<port>/swagger
```
